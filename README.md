## Install and run
I assume the user has Python installed.
To setup:
``` bash
# Clone the repository
git clone https://github.com/VolodymyrVakhniuk/Pacman
cd Pacman/src

# Open the terminal and install
pip3 install PyOpenGL PyOpenGL_accelerate numpy glfw pyrr
```
To run:
``` bash
python3 main.py
```
