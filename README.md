## Install
``` bash
# Clone the repository
$ git clone https://github.com/VolodymyrVakhniuk/Pacman

# Open the terminal and install
$ pip3 install PyOpenGL PyOpenGL_accelerate
$ pip3 install numpy
$ pip3 install glfw
$ pip3 install pyrr
```
